# Installation module
***  
__Pre-installation:__  
1. `npm i discord.js`
2. `ffmpeg installed` __correctly__
3. `npm i fs`
4. `npm i path`
5. `npm i process`
6. `npm i colors`
7. `npm i moment`
8. `npm i console-stamp`
9.  or `npm i discord.js fs path process colors moment console-stamp`
